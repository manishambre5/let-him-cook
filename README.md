# let him cook!: A handy Recipe Generator

A handy recipe generator that generates a recipe with ingredients that the user has on hand using DeepSeek.

[Frontend Demo](https://manishambre5.github.io/let-him-cook/)

Note: Mock recipe shown for a frontend demo. The backend is not deployed publicly to prevent API key exposure and excessive inference usage. In production, this would be deployed via serverless infrastructure (e.g., Vercel or AWS Lambda). A full demo is shown below...

## Tech Stack

- **Frontend:** React, TailwindCSS, react-markdown, tailwindcss-typography
- **Backend:** Node.js, Express.js, HuggingFace Inference
- **AGI Model:** DeepSeek v3.2
